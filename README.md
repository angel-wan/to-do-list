# To-do list

An application used to record to-do item, built with React, Redux, TypeScript, and CSS.

## Installation and Setup Instructions

Clone down this repository. You will need `npm` installed globally on your machine.

Installation:

`npm install`

To Start Server:

`npm run dev`

To Visit App:

`localhost:5173`
